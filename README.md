# 🎬 Movie Recommendation System

A machine learning-based recommendation system for movies, built using Python and multiple ML techniques including collaborative filtering, SVD, and GAN-based approaches.

---

## 📌 Project Overview

This project was developed as part of the Microsoft Machine Learning program under the Digital Egypt Pioneers Initiative (DEPI). It recommends movies to users based on their preferences and viewing history using multiple recommendation techniques.

---

## 📁 Project Structure

| File | Description |
|------|-------------|
| `Recommendation_System_in_Python.ipynb` | Core recommendation engine |
| `Analysis_Part.ipynb` | Data exploration and analysis |
| `SVD.ipynb` | SVD-based collaborative filtering |
| `GAN.ipynb` | GAN-based recommendation approach |
| `MLFlow_Prompt_Engineer.ipynb` | MLFlow experiment tracking |
| `streamlit.py` | Web app interface using Streamlit |
| `Project Report.pdf` | Full project documentation |
| `Recommendation System Presentation.pdf` | Project presentation slides |

---

## 🛠️ Technologies Used

- Python
- Pandas, NumPy
- Scikit-learn
- SVD (Singular Value Decomposition)
- GAN (Generative Adversarial Network)
- MLFlow
- Streamlit

---

## 🚀 How to Run

1. Clone the repository
   git clone https://github.com/GannaTaher/Recommendation_System_for_Movies.git

2. Install dependencies
   pip install -r requirements.txt

3. Run the Streamlit app
   streamlit run streamlit.py
